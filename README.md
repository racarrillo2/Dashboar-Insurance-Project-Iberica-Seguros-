# 📊 Proyecto de Analítica de Seguros — Microsoft Fabric & Power BI

## 🚀 Descripción General del Proyecto

Este proyecto fue desarrollado utilizando **Microsoft Fabric** y **Power BI** con el objetivo de analizar el rendimiento de una compañía de seguros mediante dashboards ejecutivos y operativos.

El propósito principal es transformar datos brutos del negocio en información estratégica que permita mejorar la toma de decisiones enfocadas en:

- Retención de clientes
- Análisis de rentabilidad
- Rendimiento de pólizas
- Monitoreo de ingresos
- Seguimiento de KPIs ejecutivos
- Detección de fuga de clientes (churn)

El informe fue diseñado para ofrecer una experiencia visual intuitiva, moderna y orientada al negocio.

---

# 🛠️ Tecnologías Utilizadas

| Tecnología | Uso |

| Microsoft Fabric | Integración y análisis de datos |

# 📈 Principales Insights del Negocio

## 1. Dashboard Ejecutivo
Enfocado en indicadores clave para la toma de decisiones:

- Monitoreo de ingresos totales

- Distribución de clientes
- Evolución mensual del rendimiento

## 2. Análisis de Rentabilidad y Churn
Orientado a detectar riesgos y oportunidades de mejora:

- Comportamiento de fuga de clientes
